# AI Usage Declaration

AI assistance was used during development of this project.

## Tools Used

- AI-assisted development tools were used to support code editing, debugging, documentation, and UI refinement.
- AI assistance was used for responsive UI styling, FastAPI route scaffolding, API integration structure, README updates, and error troubleshooting.

## Human Role

- The project owner provided requirements, API credentials, design direction, product links, and feedback.
- The project owner reviewed outputs and requested changes throughout development.

## AI-Generated Areas

- React page/component structure and styling.
- FastAPI route and service scaffolding.

- Google OAuth integration structure.
- Documentation drafts.

## Notes

- Secrets and API keys are expected to remain in local `.env` files only.
- AI output should be reviewed before production deployment, especially authentication, payment/product links, and astrological recommendation logic.
